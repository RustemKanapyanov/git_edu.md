# Инструкция по командной строке 

## Листинг файлов в текущей директории 
Windows 
```sh
dir
```
Linux, MacOs
```sh
ls
```

## Сменить директорию 
Windows
```sh
cd C:\Users\Name\FolderName
```
Linux, MacOs
```sh
cd/Users/Name/FolderName
```

## Создать директорию
```sh
mkdi NewFolder
```
## Удалить файл 
Windows 
```sh
del FileName 

Linux, MacOs
```sh
rm FileName
```
